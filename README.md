# 项目名称
Scan

## 项目简介
Scan 是一款强大高效的网络安全工具，旨在帮助用户进行信息收集、扫描与渗透测试。

## 功能特色
- 功能 1：[metasploit调用]
- 功能 2：[whois查询]
- 功能 3：[md5解密]

## 安装要求
- 操作系统：
  - Linux（Kali Linux、Ubuntu）
  - Termux（Android 平台）
- 软件依赖：
  - [无]

## 安装步骤
### Kali Linux 和 Ubuntu
1. 更新软件包列表：
   ```bash
   sudo apt update
   ```
2. 安装[所需依赖]：
   ```bash
   sudo apt install [package_name]
   ```
3. 克隆仓库：
   ```bash
   git clone https://github.com/huocai250/Scan.git
   ```
4. 进入项目目录：
   ```bash
   cd Scan
   ```
5. 运行应用程序：
   ```bash
   python main.py
   ```

### Termux（Android 平台）
